# ClaudeForge Documentation

Welcome to the ClaudeForge documentation. ClaudeForge is an open, no-auth plugin marketplace for Claude Code. Browse and install plugins via the web UI or CLI, publish your own, and extend Claude Code's capabilities.

## Quick Navigation

- **[Getting Started](./getting-started.md)** — Browse the catalog, search for plugins, install via CLI, and configure your setup. Start here if you're new.

### For Plugin Authors

## Key Concepts

### Plugin Types
- **Skill** — Standalone functions that extend Claude Code's capabilities
- **Hook** — Code that runs at specific lifecycle points
- **Agent** — Long-running or multi-step automation
- **Command** — CLI or slash commands
- **Plugin** — General-purpose bundles

### Supported Languages
- TypeScript
- Python
- Go
- Rust
